# 外链跳转中间页

::: tip 推荐
建议开启。保护站点权重，避免外链传递 SEO 权重。
:::

## 功能说明

点击文章中的外部链接时，先跳转到中间页面，再跳转到目标站点。支持多种平台样式：

| 平台 | 说明 |
|------|------|
| 知乎 | 知乎风格引导页 |
| 腾讯文档 | 腾讯风格引导页 |
| 石墨文档 | 石墨风格引导页 |
| 简书 | 简书风格引导页 |
| CSDN | CSDN 风格引导页 |
| 微信社群 | 微信社群引导页 |
| 少数派 | 少数派风格引导页 |
| WPS | WPS 风格引导页 |

## 使用场景

- **SEO 优化**：避免直接外链传递权重，使用 `rel="external nofollow"`
- **安全提示**：提醒用户即将离开当前站点
- **品牌展示**：中间页可展示站点品牌信息
- **流量统计**：可统计外链点击数据

## 开启方法

1. 进入 **魔法工具箱** → **页面功能**
2. 开启「外链跳转中间页」
3. 选择跳转样式（知乎/腾讯/石墨等）
4. 保存配置

## 配置选项

| 选项 | 说明 | 默认值 |
|------|------|--------|
| 跳转样式 | 中间页展示风格 | 默认 |

## 技术实现

- 使用正则匹配文章内容中的 `<a>` 标签
- 自动替换 `href` 为 `go_to/?url=编码后URL`
- 通过 `template_redirect` 钩子拦截请求
- 使用 `register_activation_hook` 注册 rewrite rule

## 注意事项

- 仅替换文章内容和评论中的链接，不影响其他区域
- 图片链接不会被替换
- 站内链接不会被替换
- 新安装站点自动注册 rewrite rule，升级站点需刷新固定链接
