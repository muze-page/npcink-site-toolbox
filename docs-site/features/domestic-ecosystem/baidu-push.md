# 百度推送

::: tip 推荐
建议开启。加速百度收录。
:::

## 功能说明

将文章链接主动推送给百度搜索引擎，加快收录速度。支持三种推送模式：

| 模式 | 说明 | 触发时机 |
|------|------|----------|
| 主动推送 | 发布/更新文章时立即推送 | 文章发布/更新 |
| 自动推送 | 页面加载时自动推送 JS 代码 | 页面访问 |
| 批量推送 | 手动选择历史文章批量推送 | 手动操作 |

## 使用场景

- 新站快速被百度收录
- 更新文章后主动通知百度
- 批量推送历史未收录文章
- 替代 Google Search Console 的 URL 检查功能

## 开启方法

1. 进入 **魔法工具箱** → **国内生态**
2. 开启「百度推送」
3. 填写站点地址和推送 Token
4. 选择推送模式
5. 保存配置

## 配置选项

| 选项 | 说明 | 必填 |
|------|------|------|
| 站点地址 | 百度站长平台中的站点 | ✅ |
| Token | 百度推送接口 Token | ✅ |
| 主动推送 | 发布时自动推送 | - |
| 自动推送 | 页面加载时推送 | - |
| 批量推送 | 手动批量推送历史文章 | - |

## 获取 Token

1. 登录 [百度站长平台](https://ziyuan.baidu.com/)
2. 进入「数据引入」→「链接提交」
3. 选择「主动推送（实时）」
4. 复制接口中的 Token 参数

## 注意事项

- 主动推送和自动推送可同时开启
- 批量推送适合新站初始化时使用
- 推送失败会在后台显示错误信息
- 每日推送配额有限，请合理使用
