# OSS 对接

::: tip 推荐
大流量站点建议开启。节省服务器存储和带宽。
:::

## 功能说明

将 WordPress 媒体文件自动上传到对象存储服务，支持阿里云 OSS、腾讯云 COS、七牛云。上传后媒体文件 URL 自动替换为 CDN 地址。

## 支持的 Provider

| Provider | 说明 | 适用场景 |
|----------|------|----------|
| 阿里云 OSS | 阿里云对象存储 | 国内站点首选 |
| 腾讯云 COS | 腾讯云对象存储 | 微信生态用户偏好 |
| 七牛云 | 七牛云存储 | 免费额度充足 |

## 使用场景

- 服务器存储空间不足
- 媒体文件访问量大，需要 CDN 加速
- 多服务器部署，需要共享媒体文件
- 降低服务器带宽成本

## 开启方法

1. 进入 **魔法工具箱** → **国内生态**
2. 开启「OSS 对接」
3. 选择 Provider
4. 填写 AccessKey、SecretKey、Bucket、Region
5. 配置 CDN 域名
6. 保存配置

## 配置选项

| 选项 | 说明 | 必填 |
|------|------|------|
| Provider | 对象存储服务商 | ✅ |
| AccessKey | 访问密钥 ID | ✅ |
| SecretKey | 访问密钥 | ✅ |
| Bucket | 存储空间名称 | ✅ |
| Region | 服务区域 | ✅ |
| CDN 域名 | 自定义 CDN 域名 | ✅ |
| 删除本地文件 | 上传后删除本地文件 | - |

## 注意事项

- 开启后仅影响新上传的文件
- 已有文件需手动迁移或使用同步工具
- 建议先测试上传后再正式启用
- 删除本地文件选项需谨慎开启
- 确保 CDN 域名已配置 SSL 证书

## 获取密钥

### 阿里云 OSS
1. 登录阿里云控制台
2. 进入 AccessKey 管理
3. 创建 AccessKey（建议使用子账号）

### 腾讯云 COS
1. 登录腾讯云控制台
2. 进入访问管理 → API 密钥管理
3. 创建密钥（建议使用子账号）

### 七牛云
1. 登录七牛云控制台
2. 进入密钥管理
3. 查看 AccessKey/SecretKey
