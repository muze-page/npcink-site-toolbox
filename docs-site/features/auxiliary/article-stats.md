# 文章统计

::: tip 推荐
内容分析建议开启。了解文章阅读情况。
:::

## 功能说明

统计文章阅读量，提供可视化图表展示。支持今日数据、月度数据和近期趋势展示。

## 使用场景

- 了解哪些文章最受欢迎
- 分析内容表现趋势
- 优化内容策略
- 展示热门文章

## 开启方法

1. 进入 **魔法工具箱** → **辅助功能**
2. 开启「文章统计」
3. 保存配置

## 数据展示

- **今日数据**：今日访问量、热门时段
- **月度数据**：本月访问趋势
- **近期趋势**：近 7 天访问量对比

## 技术实现

- 使用 `$wpdb` 直接查询数据库
- 单次 GROUP BY 查询，避免 N+1 问题
- 数据缓存在内存中，减少数据库压力

## 注意事项

- 统计基于 Cookie，同一用户多次访问会重复计数
- 如需精确统计，建议对接 Google Analytics
- 统计数据不区分搜索引擎爬虫
