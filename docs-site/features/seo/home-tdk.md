# 首页 TDK

::: tip 推荐
SEO 基础功能，必须开启。
:::

## 功能说明

自定义首页的 Title、Description、Keywords（TDK），替代 WordPress 默认的站点标题和描述。

## 使用场景

- 优化首页搜索引擎排名
- 自定义首页在搜索结果中的展示
- 设置针对性的关键词

## 开启方法

1. 进入 **魔法工具箱** → **SEO 功能**
2. 开启「首页 TDK」
3. 填写标题、关键词、描述
4. 保存配置

## 配置选项

| 选项 | 说明 | 建议 |
|------|------|------|
| 标题 | 首页 `<title>` 标签 | 包含核心关键词，不超过 30 字 |
| 关键词 | `<meta keywords>` | 3-5 个核心关键词，逗号分隔 |
| 描述 | `<meta description>` | 150 字以内，吸引点击 |

## SEO 建议

- 标题包含核心关键词，放在靠前位置
- 描述具有吸引力，提高搜索点击率
- 关键词不要堆砌，自然排列
- 定期根据搜索数据优化 TDK

## 注意事项

- 开启后覆盖 WordPress 默认首页标题
- 主题如已有 TDK 功能可能冲突
- 修改后需要等待搜索引擎重新抓取
