# 用户反馈

::: tip 推荐
建议开启。收集用户意见，改进产品体验。
:::

## 功能说明

内置用户反馈系统，支持反馈提交、设置管理和数据洞察。

## 功能模块

### 提交反馈

用户可提交以下类型的反馈：
- **功能建议**：建议新功能或改进现有功能
- **Bug 报告**：报告使用中遇到的问题
- **使用问题**：咨询使用方法
- **其他**：其他类型的反馈

反馈自动附带环境信息（WordPress 版本、PHP 版本、主题等）。

### 反馈设置

| 选项 | 说明 |
|------|------|
| 启用反馈功能 | 总开关 |
| 启用反馈表单 | 是否展示反馈表单 |
| 反馈接收邮箱 | 接收反馈通知的邮箱 |
| 自动回复 | 自动回复内容 |

### 数据洞察

- 反馈类型分布统计
- 功能开启率排行
- 活跃用户估算

## 开启方法

1. 进入 **魔法工具箱** → **其他功能**
2. 开启「用户反馈」
3. 配置接收邮箱
4. 保存配置

## 匿名数据收集

插件支持匿名数据收集，用于：
- 统计功能开启率
- 分析用户使用习惯
- 指导产品迭代方向

所有数据完全匿名，不收集任何个人信息。

## 注意事项

- 反馈功能依赖 WordPress REST API
- 确保站点可正常访问 REST 接口
- 反馈数据存储在本地数据库
