# 工单系统

::: tip 推荐
适合提供技术支持的站点。
:::

## 功能说明

内置工单管理系统，支持用户提交工单、管理员处理工单。

## 功能特性

- 用户提交工单（标题、描述、分类）
- 管理员后台回复处理
- 工单状态管理（待处理/处理中/已解决/已关闭）
- 工单列表和搜索

## 开启方法

1. 进入 **魔法工具箱** → **页面功能**
2. 开启「工单系统」
3. 保存配置

## 使用场景

- 提供技术支持服务
- 处理用户问题反馈
- 跟踪问题处理进度
- 建立服务记录

## 注意事项

- 工单数据存储在自定义数据表中
- 卸载插件时自动清理工单数据
- 建议定期清理已关闭的工单
