# v2.6 产品化稳定版收口总结

## 背景

v2.6 不继续堆功能，目标是把 WP Magick Toolbox 从“90+ 功能集合”收口为更适合中国 WordPress 站长使用的稳定工具箱版本。本轮主攻新手配置、国内访问适配、发布阻塞修复、高风险功能治理和诊断反馈闭环。

## 本次完成

- 修复百度推送 REST API 缺失回调，补齐 `rest_batch_push`。
- 修复 SVG 安全清洗，覆盖 `javascript:`、`vbscript:`、`expression(` 等风险。
- 修复搜索增强、短代码运行器等 PHP 语法问题。
- 新增首次配置向导，提供个人博客、企业官网、内容 SEO 站 3 个场景。
- 首次配置向导改为 diff 确认、快照、统一保存、刷新配置后再完成。
- 新增中国访问适配检测与一键修复建议，Gravatar、Google Fonts、Google Ajax 变更需 diff 确认后保存。
- 数据库清理、数据库导出等移入高风险层级；SVG 上传移入进阶层级。
- 数据库清理改为先预览、按类型门控、确认后显式 `dry_run: false` 执行。
- 诊断中心增加中国访问适配项，诊断报告导出增加敏感信息脱敏。
- README、readme.txt、docs-site changelog 同步到 2.6.0。
- PHPUnit risky 清理完成，测试在 PHP 8.5 下无 failure、无 risky。

## 验证结果

已通过：

```bash
find . -path './vendor' -prune -o -path './vite/*/node_modules' -prune -o -path './vite/*/dist' -prune -o -name '*.php' -print0 | xargs -0 -n1 php -l
vendor/bin/phpunit --configuration phpunit.xml.dist
npm run --prefix vite/admin test:run
npm run --prefix vite/admin build
```

结果摘要：

- PHP 全量语法检查通过。
- PHPUnit：`OK (204 tests, 1134 assertions)`。
- Vitest：`25 tests passed`。
- Vite build 成功。

保留的非阻塞警告：

- Vite `base option should start with a slash`。
- Vite chunk size warning。

## 当前判断

v2.6 已达到代码层面的发布门槛。下一步应进入打包前人工验收，不建议继续加入 v2.7 新功能。

## 打包前人工验收清单

- 首次配置向导：选择方案、取消 diff、确认 diff、保存后刷新配置。
- 国内环境适配：检测、生成修复建议、取消 diff、确认 diff、保存后刷新配置。
- 百度推送 REST：缺配置、传入 URL、按 offset 批量推送三种路径。
- 数据库清理：预览后执行，确认请求包含 `dry_run: false`。
- 诊断报告导出：确认 API Key、Secret、Token、邮箱、IP 等敏感字段已脱敏。

## 后续不纳入 v2.6 的事项

以下进入 v2.7 或更后版本，不阻塞本次发布：

- 搜索健康中心。
- 验证码中心。
- CDN 刷新。
- WebP/AVIF。
- SMTP。
- 更细粒度的前端代码分包优化。
